Repo: Jekyll Sitemap Generator Github

  1. Add gem 'jekyll-sitemap' to your Gemfile
  2. Run bundle
  3. Add - jekyll-sitemap to your _config.yml under gems:
  4. Run jekyll build to reload your config and sitemap.xml so be in _site
